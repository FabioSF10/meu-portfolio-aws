01. API de IA Generativa

Integração Serverless Híbrida (AWS & Azure)

Este projeto demonstra, na prática, a construção de uma arquitetura híbrida entre AWS e Azure para entrega de inteligência artificial em tempo real.

  • Gateway de API: Endpoint público exposto via AWS API Gateway, garantindo segurança e controle de requisições.
  • Orquestração Serverless: AWS Lambda responsável por receber, validar e processar as mensagens do usuário antes de encaminhar para o modelo de IA.
  • Integração Multi-Cloud: Comunicação direta com o Azure OpenAI Service, consumindo modelos LLM para geração de respostas inteligentes.
  • Contexto Inteligente (RAG leve): A IA é enriquecida com informações reais do meu perfil profissional, permitindo respostas personalizadas sobre minha experiência, projetos e carreira.
  • Arquitetura: Sistema desacoplado, escalável e com baixo custo operacional, seguindo boas práticas de Cloud Computing.
